# cherv-say-python
## Описание
Аналог CowSay, только вместо коровы - червяк.
## Установка и использование
Установка:
```bash
pip install cherv
```
Ипользование
```bash
python -m cherv [сообщение]
```
Сообщение можно вводить любой длины:
